# Uniter 

------
## Unit converter
**Current version:** *v1.0.0*

------
## Using from terminal:
**$ python3 cli.py --type=**_weight_ **--main=**_g_ **--dest=**_kg_ **--value=**_123.45_<br>
**$ 0.12345**

###### Value can be *float* and *int* type

## Available types:
- **length:**
  - mm
  - cm
  - m
  - km
- **weight:**
  - g
  - kg
